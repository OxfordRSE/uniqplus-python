Snakes live within ecosystems, and so does the Python programming language. In this case , the ecosystem is made of only three components:

  • Python itself, that is the program that interprets the language and execute programs.
  • A collection of packages, which are python programs written by others (or maybe yourself) that you can reuse in your own Python programs.
  • A package manager, which is a piece of software which allows you to install, updates and remove packages. There are two main package managers: pip and conda. In this course we will mainly be using conda, but we'll mention pip as well.

This document describes the installation of Anaconda, which is a Python distribution. This is a bundle that contains Python, the conda package manager and a large collection of packages, so you don't have to install everything yourself.

1. Step 1: Install Anaconda

1.1. Windows
1.1.1. Download

Download Anaconda Individual Edition at https://www.anaconda.com/products/individual

1.1.2. Installation

Double click on the downloaded installer (`Anaconda3-2021.05-Windows-x8664.exe`) and follow the steps. You can go through all the steps by accepting the default settings, as shown below.

A detailed walk-trough the installation of Anaconda is available at

https://docs.anaconda.com/anaconda/install/windows/

Step 2 "Verify data integrity with SHA-256" can be skipped.

1.2. MacOS
1.2.1. Download

You can download Anaconda at

https://www.anaconda.com/products/individual#Downloads

Choose the "64-bit Graphical Installer".

1.2.2. Installation

Double-click the downloaded file and click continue to start the installation

You can then follow the steps by accepting the default settings. You do not need to install the PyCharm IDE.

A detailed walktrough the installation of Anaconda is available at

https://docs.anaconda.com/anaconda/install/mac-os

Step 2 "Verify data integrity with SHA-256" can be skipped.

1.3. GNU/Linux
1.3.1. Download

You can Download Anaconda at

https://www.anaconda.com/products/individual/Downloads

Choose the "64-Bit (x86) Installer"

1.3.2. Installation

You can run the downloaded installer by entering the following command in a terminal 

bash ~/Downloads/Anaconda3-2021.05-Linux-x86_64.sh

This is assuming your downloaded the installer in the Downloads directly.

You can then follow the steps by accepting the default settings. In particular, you should answer yes to "Do you wish the installer to initialise Anaconda3 by running conda init?". You do not need to install the PyCharm IDE.

A detailed walktrough the installation of Anaconda is available at

https://docs.anaconda.com/anaconda/install/linux

Step 2 "Verify data integrity with SHA-256" can be skipped.

1.4. Disabling auto-activation of conda (MacOS and GNU/Linux) (optional)

By default, the conda is automatically activated whenever you start a new terminal (a new shell). This can be handy but you can also choose to deactivate this by running the following conda command in a terminal (on MacOS: Launchpad -> Terminal) 

conda config --set auto_activate_base False

In this case, you'll have to manually activate conda by running 

conda activate base

2. Step 2: Verify your installation

2.1. Windows

Open Anaconda prompt from the start menu:

Type conda list and press ENTER. You should see a list of all the installed packages.

You can start Python by running the python command in the Terminal. Type quit() to exit Python.

2.2. MacOS

Open the Terminal application (Launchpad -> Terminal) and type conda list then ENTER. You should see a long list of all the Python packages installed.

You can start Python by running the python command in the Terminal. Type quit() to exit Python.

2.3. GNU/Linux
  1. Open a terminal

  2. Provided that the conda environment is activated, i.e. the command prompt does start with (base) (see 1.4), type conda list and press ENTER. You should see a list of all the installed packages.

    You can start Python by running the python command in the Terminal. Type quit() to exit Python.

3. Step 3: Update Anaconda and install spyder-kernels

In the Anaconda Prompt (Windows) or terminal emulator (GNU/Linux and MacOS), enter the following commands. For each of the commands below, type the command (for instance conda update conda) and press ENTER. Some of then will ask for your permission to carry one, in this case press the y key and then press ENTER.

On MacOS, you can start a terminal emulator by running the Terminal app (Launchpad -> Terminal)

  1. Update the conda package manager 

    conda update conda

  2. Update the pre-installed Python packages 

    conda update anaconda

  3. Finally, install the spyder-kernels package. This will allow you to interact with a Python session directly from the spyder editor. This is very useful when writing Python programs! 

    conda install spyder-kernels
